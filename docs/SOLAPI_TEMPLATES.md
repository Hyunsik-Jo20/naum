# 솔라피(Solapi) 알림톡 템플릿 · 발신번호 승인 제출용

> 나음(NaUM) 보건실 알림 발송용. **이 문서의 템플릿을 솔라피 콘솔에 그대로 등록** → 카카오 심사 제출.
> 앱 실제 발송 문구(`src/data/notifyText.ts`) 구조와 1:1로 맞춤. 최초 작성: 2026-07-10.

## 0. 승인이 필요한 것 (2가지)
1. **발신번호 등록** (SMS/LMS·알림톡 대체발송용)
   - 솔라피 콘솔 → 발신번호 등록 → 학교 대표번호 또는 보건실 번호.
   - 서류: **통신서비스 가입증명원**(해당 번호) 또는 재직/사업자 증빙. 학교(기관)는 기관 명의 증빙.
   - 심사가 아니라 **본인/기관 확인 절차**(보통 1~2영업일).
2. **카카오 알림톡 템플릿 심사**
   - 먼저 **카카오 비즈니스 채널** 개설 → 솔라피에 발신프로필(채널) 연결 → 아래 템플릿 등록 → **카카오 심사(승인/반려)**.
   - 심사 포인트: ① 정보성 메시지일 것(광고·혜택·이벤트 문구 금지) ② 변수(`#{}`)만으로 구성 금지(고정 안내문 필수) ③ 특수문자 과다 지양 ④ 1,000자 이내.
   - 아래 템플릿은 위 기준을 만족하도록 **결과별로 분리 + 고정 안내문 포함**.

## 1. 변수 목록 (`#{변수명}`)
| 변수 | 의미 | 예시 |
|---|---|---|
| `#{학교명}` | 학교 이름 | 테스트초등학교 |
| `#{학생명}` | 학생 이름(학부모 대상) | 장지호 |
| `#{학년반}` | 학년·반(담임 대상) | 3학년 1반 |
| `#{증상}` | 증상(증상타일 텍스트) | 두통 · 어지러움 |
| `#{병명}` | 추정 병명(보건교사 확인) | 긴장성 두통 |
| `#{처치}` | 시행한 처치 목록 | 안정, 수분 섭취 |
| `#{보건실연락처}` | 보건실/학교 문의번호 | 051-000-0000 |

> ※ 알림톡은 학부모 휴대폰으로 직접 가므로 **이름·병명이 메시지에 포함**됩니다. (앱의 익명 relay와 별개 경로 — 발송 단계에서만 로컬 PII 사용, 서버 저장 없음. 구현 시 `/api/sms`가 키·발신을 서버 보관하고 본문은 로컬에서 조립.)

## 2. 학부모 대상 템플릿

### T1. 보건실 접수 안내
```
[#{학교명} 보건실]

#{학생명} 학생이 보건실에 접수되었습니다.

▶ 증상: #{증상}

보건교사가 살펴본 뒤 처치 결과를 다시 안내드리겠습니다.

※ 문의: 보건실 #{보건실연락처}
```

### T2. 처치 종료 안내 — 교실 복귀
```
[#{학교명} 보건실]

#{학생명} 학생이 보건실 처치를 마치고 교실로 복귀했습니다.

▶ 증상: #{증상}
▶ 추정 병명: #{병명}
▶ 시행한 처치: #{처치}

처치 후 안정되어 교실로 복귀했습니다. 가정에서도 상태를 살펴봐 주세요.

※ 문의: 보건실 #{보건실연락처}
```

### T3. 처치 종료 안내 — 귀가
```
[#{학교명} 보건실]

#{학생명} 학생이 보건실 처치 후 귀가 조치되었습니다.

▶ 증상: #{증상}
▶ 추정 병명: #{병명}
▶ 시행한 처치: #{처치}

안정이 필요해 귀가 조치되었습니다. 가정에서 충분히 쉬게 해주시고, 증상이 지속되면 진료를 받아보세요.

※ 문의: 보건실 #{보건실연락처}
```

### T4. 처치 종료 안내 — 병원 이송
```
[#{학교명} 보건실]

#{학생명} 학생이 추가 진료가 필요해 병원으로 이송되었습니다.

▶ 증상: #{증상}
▶ 추정 병명: #{병명}
▶ 시행한 처치: #{처치}

자세한 사항은 보건실로 연락 부탁드립니다.

※ 문의: 보건실 #{보건실연락처}
```

### T5. 보건실 관찰 안내
```
[#{학교명} 보건실]

#{학생명} 학생을 보건실에서 관찰하고 있습니다.

▶ 증상: #{증상}
▶ 추정 병명: #{병명}
▶ 시행한 처치: #{처치}

보건실에서 경과를 관찰하고 있습니다. 변화가 있으면 다시 안내드리겠습니다.

※ 문의: 보건실 #{보건실연락처}
```

## 4. SMS/LMS 대체발송 문구 (알림톡 실패 시 자동 대체)
알림톡이 실패(채널 미등록·수신거부·앱 미설치)하면 솔라피가 **대체발송**으로 SMS/LMS를 보냅니다. 본문은 위 알림톡과 동일하게 조립하되, 90바이트 초과 시 LMS로 자동 전환됩니다. 별도 심사 없음(발신번호만 등록되면 됨).

**단문(SMS) 예시 — 접수:**
```
[#{학교명} 보건실] #{학생명} 학생이 보건실에 접수되었습니다. 증상: #{증상}. 처치 후 다시 안내드립니다. 문의 #{보건실연락처}
```

## 5. 등록 순서 (콘솔 작업)
1. 솔라피 가입 → **발신번호 등록**(서류 제출) → 승인 대기.
2. 카카오 비즈니스 → **채널 개설** → 솔라피 발신프로필 연결.
3. 솔라피 알림톡 → **템플릿 등록**(위 T1~T10, 총 10개 복사) → 카카오 심사 제출.
   - 학부모용 5개(T1 접수 + T2~T5 종료 4결과) · 담임용 5개(T6 접수 + T7~T10 종료 4결과).
4. 승인 후 앱 `/api/sms` 연동 시 `templateId` + 변수 매핑으로 발송.
